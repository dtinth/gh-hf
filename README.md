A [GitHub CLI](https://cli.github.com) extension to help maintainers set up repo for Hacktoberfest, e.g. adding `hacktoberfest` topic and labels to a repo.

## Installation

```shell
gh extension install dtinth/gh-hf
```

## Usage

```shell
gh hf enable
```
